Ciclo de Conversas de Design e Multimédia

This project had the objective of creating the entire visual identity and website for the "Ciclo de Conversas de Design e Multimédia", which consists of a cycle of conferences that is held annually for the bachelor and master's degree in Design and Multimedia at the University of Coimbra.

Indigrow

In this project we were challenged to make an experimental film about the colour "Indigo", having as project phases the pre-production, the production and the post-production. This work was made during a pandemic, so we had to use open-source resources.

CultureInn

The "CultureInn" project consisted of a UI/UX design project, whose challenge was to create an application focused on the "wellness" theme. We wanted to create a tool to promote cultural well-being. We designed an application that fosters cultural activity and provides stimulating challenges, that works as a "cultural calorie meter". Each user has the opportunity to progress within the app, by attending events/activities of a cutural nature.

Encrypted messages

Based on the exhibition "Cybernetic Serendipity (1968)", the goal of this project was to develop an interactive multimedia object, through appropriation, reinterpretation or deconstruction. We focused on creating a language that enables interaction between at least two individuals, using cryptology as a tool to translate words into forms. We created an application in React Native that serves as a tool to facilitate the decoding and exchange of messages.
